Progen

 
 

La mejor forma de utilizar la herramienta, es integrándola en un IDE para poder desarrollar con las ventajas que ofrece utilizar este tipo de entornos.


Nuestra elección ha sido eclipse, por la cantidad de plugins que incluye, así como la facilidad de uso del mismo.


Es importante mantener una serie de normas para poder trabajar con comodidad.


  1. Configura el workspace de eclipse para que apunte a la carpeta padre de donde hayas descargado el código de subversion.

  2. Crea un nuevo proyecto Java, indicando como nombre de proyecto el nombre de la carpeta donde hayas descargado el código.

  3. Configura eclipse para que utilice la codificación de caracteres UTF-8. Para ello, selecciona el proyecto recién creado y en el apartado de Resource seleccionar UTF-8 como Text file encoding.


Con esto ya estás en disposición de empezar a utilizar ProGen.
